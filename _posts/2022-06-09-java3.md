# Java 자료형

- 자료형을 지켜 소스 코드를 작성하면 메모리(RAM)의 공간 복잡도가 낮아 자원을 절약할 수 있다. 각 자료형의 메모리 크기(byte)와 표현 가능 범위를 알아두자
- 메모리 크기의 관점에서 `long a = 12345678;` = `long a = 1;` 같다.
