📌 **작성자 개발 환경** <br>
**OS** : Windows 10 <br>
**Language** : css<br>
**Tool** : Visual Studio Code<br>
{: .notice--primary}

- `<a> ~ </a>`와 `<h1> ~ </h1>` 요소를 비교해보자.
- a 태그는 줄바꿈이 없이 Text가 연속해서 자리잡고, h1 태그는 웹 페이지에서 raw에 전체를 차지한다.
- 아래 html 코드와 출력물 예시로 알아보자.

```html
<head>
    <style>
        h1 {
            border-width: 5px;
            border-color: red;
            border-style: solid;
        }
        a {
            border-width: 5px;
            border-color: red;
            border-style: solid;
        }
    </style>
</head>
<body>
    <h1>CSS</h1>This is <a href="www.google.com">CSS</a> contents
</body>
```
